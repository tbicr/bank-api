    URL obj = new URL("https://api.nbrb.by/banks");
HttpURLConnection con = (HttpURLConnection) obj.openConnection();
con.setRequestMethod("GET");
int responseCode = con.getResponseCode();
BufferedReader in = new BufferedReader(
    new InputStreamReader(con.getInputStream()));
String inputLine;
StringBuffer response = new StringBuffer();
while ((inputLine = in.readLine()) != null) {
    response.append(inputLine);
}
in.close();
System.out.println(response.toString());

    Currencies

    Currencies.

    Parameters

    Parameter In Type Required Description
    Api-Version header string false API version (default latest).
    lang query string true Language of text data.
    at query dateTime false Currancy rate at specific date. Current time used by default.

    Example responses

    {
  "items": [
    {
      "type": "string",
      "currency_from": "string",
      "currency_from_name": "string",
      "currency_to": "string",
      "currency_to_name": "string",
      "exchange_rate": "string"
    }
  ]
}

    {
  "code": "string",
  "message": "string",
  "fields": "string"
}

    Responses

    Status Meaning Description Schema
    200 OK An array of currencies. CurrencyRateList
    default Default Unexpected error Error
    This operation does not require authentication
